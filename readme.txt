Social site sharer links widget automatically places popular social share links in a widget placement without configuration.

== Description ==

Social site sharer links widget takes your site url and blog name to construct social share links in any widget placement. Simply "Click and Forget." No complicated configuration needed at all, however you can choose which ones to hide in the widget settings panel.

== Installation ==

1. Upload `social-site-sharer-links-widget` zip folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose what social media you like in widget settings

== Frequently asked questions ==

= How does this plugin differs from other =

This plugin will provide your site a sharing link to social media without any configuration needed. You can simply choose what social media you want.
